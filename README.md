# TaskApi
Projeto para gerenciamento de tarefas em relação ao seu progresso: aberto, fazendo, concluído.

## Ferramentas
* Spring Boot
* Spring Data + H2
* Spring Validation
* Swagger2

## Requisitos
* Java 11
* Maven
* Eclipse Workspace

## Como executar o projeto?
